# gatsby-remark-slug

Add slug `id` to h1~h6 elements.

## Install

`npm install --save gatsby-remark-slug`

## How to use

```javascript
// In your gatsby-config.js
plugins: [
  {
    resolve: `gatsby-transformer-remark`,
    options: {
      plugins: [`gatsby-remark-slug`],
    },
  },
]
```
